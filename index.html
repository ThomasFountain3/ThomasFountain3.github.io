Current Projects

Virtual Pet

With this project being the first individual project I've ever done, it came out better than I expected. Main goal was using basic Java functions in order to develop a Rocko "The Virtual Dog" Here, you can feed, play, and give Rocko water in order to keep him happy. The goal of this is to make sure that Rocko stays happy by completing the correct task in a timely matter; just like taking care of a dog.

Virtual Pet Shelter

On this project, I used the same properties that I used on Rocko "The Virtual Dog" and I built on top of that. Here, we've created multiple different pets that someone can choose what pet they want to take home. You will be able to choose from a mulitude of pets, interact with them just like we did on the first project. The goal for this project was to show the ability to create different characters with their own individual characteristics.
